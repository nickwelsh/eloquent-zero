# Security Policy

## Supported versions

This package is early-stage. Security fixes will generally land on the latest `main` branch state and the latest released version.

## Reporting a vulnerability

Please do not open a public issue for security reports.

Instead, use GitHub's private vulnerability reporting for this repository:

- [Report a vulnerability](https://github.com/nickwelsh/eloquent-zero/security/advisories/new)

If that is unavailable, contact the maintainer privately through GitHub.

Please include:

- a description of the issue
- affected versions, if known
- reproduction steps or a proof of concept
- impact assessment

You should receive an initial response within a reasonable time. After triage, a fix and release plan will be coordinated as needed.
